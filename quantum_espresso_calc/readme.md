This directory has various subdirectories relating to Quntum Espresso calculations that contain files and output files used to perfrom DFT calculations.

    The "psuedo_potentials" subdirectroy contains the psuedo potential files used to perfrom DFT calculations within Quantum Espresso

    The "formation_energy_calc" subdirectory contains outputs from DFT on atomic calculations used for atomic references for formation energy calculations

    The "relaxed_structures" subdirectory contains outputs from DFT on relaxation energy calculations on ML proposed perovskites.
